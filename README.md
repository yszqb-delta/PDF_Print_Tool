# PDF 打印小工具说明 / PDF Booklet Tool Description

## 中文说明 Chinese Description

这是一个 pdf 打印小工具，用于将 pdf 文件变成小册子的形式。  
本程序使用 PyMuPDF 库实现主要功能。  

**原理如下：**  
1. 提取源文件内容  
2. 转成图片  
3. 横过来  
4. 按照特定的顺序排在新文件的页面上  
5. 存储  

---

## English Description

This is a small tool for printing PDFs, designed to convert PDF files into booklet format.  
The program uses the PyMuPDF library to implement its main functionality.  

**Principle:**  
1. Extract content from the source file  
2. Convert to images  
3. Rotate horizontally  
4. Arrange pages in a specific order on a new file  
5. Save the result
